 🏆 CCNA1 Certificate – Introduction to Networks
 📌 Description

Ce dépôt contient mon certificat de réussite du cours CCNA1 – Introduction to Networks (Cisco Networking Academy).
Il valide mes compétences de base en réseaux informatiques, notamment la compréhension du modèle OSI, des protocoles réseau, des adresses IP et des équipements réseau.
